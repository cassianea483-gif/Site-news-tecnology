🎯 COMO USAR (RÁPIDO):

    python main.py
    
    ✓ Servidor rodando em: http://localhost:5000

2️⃣  Terminal 2 - Sirva o Frontend:
    cd frontend
    python -m http.server 8000
    
    ✓ Acesse: http://localhost:8000/index.html

3️⃣  Pronto! Use a aplicação:
    - Sign Up → Crie uma conta
    - Login → Faça login
    - Explore → Escolha categorias e veja notícias
    - ⭐ Add to Favorites → Salve artigos
    - My Favorites → Gerencie seus favoritos

🔄 FLUXO COMPLETO:

1. Usuário acessa index.html
2. Seleciona "Sign Up" → Preenche formulário
3. Dados enviados via POST /register
4. Usuário faz login → POST /login
5. user_id é salvo em localStorage
6. Frontend carrega notícias via GET /news
7. Usuário clica "Add to Favorites"
8. Favorito é salvo via POST /favorites
9. Acessa "My Favorites" para gerenciar
10. Remove favorito via DELETE /favorites
